This repository contains a collection of **Java practice programs and concepts** designed to strengthen core programming skills. It covers everything from **basic syntax to advanced features like functional interfaces, streams, and lambda expressions**.

This project is created as part of continuous learning and hands-on practice in Java. It includes:
